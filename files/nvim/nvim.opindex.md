# Neovim
Leader: Space

## Move between split windows
```
Ctrl-h
Ctrl-j
Ctrl-k
Ctrl-l
```

## Create a horizontal split
```
Ctrl-w s
```

## Create a vertical split
```
Ctrl-w v
```

## Close the current split
```
Ctrl-w c
```

## Keep only the current split
```
Ctrl-w o
```

## Search help tags
```
Space s h
```

## Search files in working directory
```
Space s f
```

## Search text in working directory
```
Space s g
```

## Search the current word across the project
```
Space s w
```

## Resume the last Telescope search
```
Space s r
```

## Search recent files
```
Space s .
```

## Search open buffers
```
Space Space
```

## Fuzzily search inside the current buffer
```
Space /
```

## Search only across open files
```
Space s /
```

## Search your Neovim config files
```
Space s n
```

## Open diagnostics in the location list
```
Space q
```

## Jump to an LSP definition
```
g d
Ctrl-t    # jump back
```

## Find LSP references
```
g r
```

## Jump to an LSP implementation
```
g I
```

## Rename the symbol under the cursor
```
Space r n
```

## Run an LSP code action
```
Space c a
```

## Search document symbols with Telescope
```
Space d s
```

## Search workspace symbols with Telescope
```
Space w s
```

## Search LSP type definitions
```
Space D
```

## Jump between git hunks
```
[ c
] c
```

## Stage or reset the current git hunk
```
Space h s
Space h r
```

## Preview a git hunk or blame the current line
```
Space h p
Space h b
```
